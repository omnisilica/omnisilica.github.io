YouTube Playlist Tracker Service

The purpose of this extension is to allow you to keep track of the videos that play while your YouTube playlist is playing. It's a simple alternative to anyone who happens to find themselves recording each video that their playlist goes through, especially on shuffle.

The Service Provider

The entity providing this service can be referred to as Omnisilica. Omnisilica is not a corporation or entity that conducts trade, currently. Rather, one can think of Omnisilica as a developer contributing to open source initiatives.

Privacy Policy

Information Omnisilica collects
In order for this extension to work we need to collect certain data that your browser produces. None of this information, however is personal in the traditional sense, i.e., email, address, etc. Rather, these are identifiers such as tab id and url parameters.

Why Omnisilica collects data
This extension uses tab id for rudimentary state management, and the url of the page you're on to determine what part of it's UI to display. If the extension detects a page outside of YouTube the UI will convey this information. Likewise, if a user detects a YouTube page without an active playlist on it, it will notify the user.
